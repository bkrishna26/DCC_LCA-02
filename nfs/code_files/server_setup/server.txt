NFS SERVER SETUP (Step-by-Step)

1. Install NFS Server
   sudo apt update
   sudo apt install nfs-kernel-server -y

2. Create Shared Directory
   sudo mkdir -p /srv/nfs/shared

3. Set Permissions
   sudo chmod -R 777 /srv/nfs/shared
   sudo chown nobody:nogroup /srv/nfs/shared

4. Configure Exports File
   sudo nano /etc/exports

Add the following line:
/srv/nfs/shared *(rw,sync,no_subtree_check,no_root_squash)

Save and exit:
CTRL + X → Y → Enter

5. Apply Changes
   sudo exportfs -a

6. Restart and Enable NFS Service
   sudo systemctl restart nfs-kernel-server
   sudo systemctl enable nfs-kernel-server

7. Verify Export
   sudo exportfs -v

8. Get Server IP Address
   ip a

Note down the IP (example: 192.168.1.10)
